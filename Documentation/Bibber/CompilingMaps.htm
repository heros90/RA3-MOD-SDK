Compiling Maps

Maps are compiled with the "BuildMap.bat." This is a completely rewritten batch program to do more things automatically an minimize user actions like copying files manually etc.
The usage of the batch file with the available parameters is explained below.

Usage
You can either just double-click the file and enter mapname and parameters manually or you can pass it through the commandline using this way:
BuildMap.bat [MapName [/cbc] [/csc] [/cbm] [/ma] [/nf]]

Parameters
/cbc: Clear build cache
/csc: Clear session cache
/cbm: Clear built map
/ma: Merge assets
/nf: Do not fix assets

MapName
The mapname is the folder name of your map in "%appdata%\Red Alert 3\Maps". The "map.xml" in this folder is the file that is being compiled. If you want to add custom assets or modifications, include it to the "overrides.xml".

Clear Build Cache
The build cache stores all compiled art and audio assets. If you compile your map the first time the assets are copied to that cache. If you now remove e.g. a model from your map and want to add it back later, it doesn't need to be compiled again. It is copied from that cache which greatly reduces the duration of the whole compiling process.
Use this parameter if you want to clear the build cache.

Clear Session Cache
The session cache stores all compiled assets of your previously compiled map. If you now remove e.g. a model or any other asset from your map, it will be removed from that cache, too. If you just change an asset, it will be compiled and updated in the cache and if nothing changed, the asset won't be compiled which saves time, too.
Use this parameter if you want to clear the session cache.

Clear Built Map
All compiled and copied files go to "BuiltMods\Mods\Maps\[MapName]". The complete content (besides *.asset) will be copied back to "%appdata%\Red Alert 3\Maps\[MapName]".
Use this parameter to delete all files from this folder.

Merge Assets
Asset files that have been extracted with the C&C Asset Extractor can be reused in your map without decompiling them. Asset and cdata files go to "%appdata%\Red Alert 3\Maps\[MapName]\Assets".
Use this parameter to merge these assets into the map stream.

Do Not Fix Assets
If you use assets in your map that refer to data of the World Builder stream (e.g. civilian structures), the models of these assets might become invisible ingame. This is being fixed when compiling the map.
Use this parameter to not fix these assets.
