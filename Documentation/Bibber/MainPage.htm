RA3 Mod SDK Expansion

The RA3 Mod SDK Expansion is a pack of tools and files to increase the functionality and productivity of the RA3 Mod SDK. It also provides x64 support and fixes some bugs with mods compiled with the original SDK.

You will get a list of options for compiling mods and maps and short explanations of them here. If you find bugs or errors of this expansion, please contact me.
