Sage Engine

Red Alert 3 was created using Electronic Art’s (EA) Sage Engine. 

The Sage Engine utilizes 3DSMax as its main application for creating art.  There are 4 main tools that are used in 3DSMax: .W3X Exporter, w3d Skin modifier, w3d utilities and w3d materials. These are generally referred to as the W3D tools.

The W3X viewer allows the user to view the W3X files exported from max before bringing them in-game.

The Sage Engine also uses WorldBuilder, which is a proprietary application for the creation of single-player and multiplayer maps for the game. An updated version of WorldBuilder is required for creating maps which depend on content in a Red Alert 3 mod, and can be downloaded at Command and Conquer.com

The Red Alert 3 Mod SDK requires certain applications to be installed for it to function properly. Please view the Required Applications page for more information.
